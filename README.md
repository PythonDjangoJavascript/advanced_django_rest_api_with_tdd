## Installation

```
$ git clone https://github.com/PythonDjangoJavascript/advanced_django_rest_api_with_tdd.git
$ cd (enter the downloaded directory)
$ docker build .
$ docker-compose build
$docker-compose run app sh -c "python manage.py runserver"
```
